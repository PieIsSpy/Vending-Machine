# Vending Machine
This program serves the customer a silog meal with the option to add as many add-ons as they like in one serving.

Originally finished in 11/23/2024

### Features
- Silog Vending Feature
  - Inserting Currency Denominations
  - Ordering a Silog Meal (Rice and Egg + Add-Ons)
- Maintenance Feature
  - Inventory
    - Viewing Inventory
    - Modifying Price
    - Restocking
  - Cash Register
    - Viewing Denomination Stock
    - Restocking Denominations
    - Cashing Out

### Prerequisites
In order to be able to run this program, you will need to install the `gcc` compiler

### Compiling
To compile the project, run:
```
gcc Main.c -o Main.exe
```

### Running
Run `Main.exe` file
